# On-Call Runbook

## Escalation Policy
- Page the primary on-call engineer immediately for P0 incidents.
- If no acknowledgment within **15 minutes**, escalate to the secondary on-call.
- If still no acknowledgment within 30 minutes, escalate to the manager on duty.

## Incident Updates
- Post an update in the incident channel every 15 minutes for P0.
- Include impact, mitigation, and next steps.

## Severity Levels
- P0: Customer-facing outage or data loss risk
- P1: Major feature broken, partial outage
- P2: Minor issues, degraded performance
